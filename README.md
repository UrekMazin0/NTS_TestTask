# Текст задания: 
```
Написать программу Справочник товаров:

Программа должна иметь следующие возможности, работая со всей информацией в единой таблице на экране пользователя:
добавление товара
редактирование товара
удаление товара
просмотр списка товара
поиск товаров по полям: Code, Name, BarCode, Price.
Автоматическое заполнение таблицы новыми строками в размере 1000 строк. 

База данных должна быть оформлена
двумя связанными таблицами со связью один ко многим 
(одно значение таблицы цен может быть заполнено у нескольких значений таблицы товаров)

Таблица цен должна содержать следующие поля:
Id - Guid
Price – Decimal

Таблица товаров должна содержать следующие поля:
Id – Guid
IdPrice - Guid
Code - Integer
Name - String
BarCode - String
Quantity - Decimal
Model - String
Sort - String
Color - String
Size - String
Weight - Decimal
DateChanges - DateTime

Требования по используемым технологиям и фреймворкам:
.Net Framework >= 4.0 
WPF 
в качестве хранилища данных можно использовать файлы XML, JSON, sqlite (предпочтительнее sqlite).

Результаты представить в виде проекта (в исходных кодах),
готового к компиляции и откомпилированного решения. 
База данных, XML или JSON должны быть предоставлены, 
заполненные тестовыми данными (не менее 1 000 строк таблицы товаров).
```
